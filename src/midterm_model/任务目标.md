**任务目标：**

你需要为一个类似 PUMA 的机械臂编写 URDF（Unified Robot Description Format）模型文件。你需要根据提供的示意图（几何形状）和参数表（运动学参数）来构建这个机器人。

---

### 1. 文件与环境背景

- **文件类型**：URDF Xacro (`robot.urdf.xacro`)。
- **依赖包**：模型应属于 `midterm_model` 包。
- **任务**：定义 Links（连杆）和 Joints（关节）并组装成完整的运动链。

### 2. 几何参数 (Links / Geometries)

你需要定义以下连杆（Links），每个连杆都有具体的几何形状和尺寸：

1.  **base_link (基座)**:

    - 形状：圆柱体 (Cylinder)
    - 尺寸：半径 $r=0.35$, 长度 $l=0.1$
    - 原点/视觉中心建议：根据表格推断，通常置于几何中心。

2.  **link_01 (第一臂/立柱)**:

    - 形状：圆柱体 (Cylinder)
    - 尺寸：半径 $r=0.15$, 长度 $l=1.1$

3.  **link_02 (第二臂/大臂)**:

    - 形状：长方体 (Box)
    - 尺寸：$0.3 \times 1.2 \times 0.2$ (长 x 宽 x 高，注意根据旋转矩阵，实际延伸方向可能对应 Y 轴或 X 轴)

4.  **link_03 (第三臂/小臂)**:

    - 形状：长方体 (Box)
    - 尺寸：$0.2 \times 1.0 \times 0.1$

5.  **effector (末端执行器)**:
    - 形状：长方体 (Box)
    - 尺寸：$0.2 \times 0.2 \times 0.1$

---

### 3. 运动学链结构 (Kinematic Chain)

这是最关键的部分，描述了关节如何连接上述连杆。请按照以下父子关系和参数生成 `joint` 定义：

**链条顺序：**
`base_link` $\rightarrow$ `J_0` $\rightarrow$ `link_01` $\rightarrow$ `J_F0` $\rightarrow$ (虚拟/中间帧) $\rightarrow$ `J_1` $\rightarrow$ `link_02` $\rightarrow$ `J_2` $\rightarrow$ `link_03` $\rightarrow$ `J_3` $\rightarrow$ `effector`

**关节详细参数表：**

- **Joint: `J_0`**

  - **类型**: Revolute (旋转关节)
  - **父连杆**: `base_link`
  - **子连杆**: `link_01`
  - **Origin (变换)**: `xyz="0 0 0.1"`, `rpy="0 0 0"`
  - **轴 (Axis)**: Z 轴 (`0 0 1`)
  - **限制 (Limits)**: Lower: `-3.14`, Upper: `3.14` (Effort/Velocity 可设为默认值，如 100/1.0)

- **Joint: `J_F0`** (关键转换关节)

  - **类型**: Fixed (固定关节)
  - **父连杆**: `link_01`
  - **子连杆**: (通常连接到一个名为 `link_02` 的连杆，但考虑到后面有 J_1，这里可能作为坐标系转换的中间节点，或者直接作为 `link_02` 的固定偏移，而 `link_02` 自身的移动由 `J_1` 驱动。但在 URDF 中，通常处理为：Link01 -> J_F0 -> Link02_Base(虚拟) -> J_1 -> Link02。**根据文档语境，更简单的理解是：J_F0 将坐标系转换到肩部位置。**)
  - **Origin (变换)**: `xyz="0 0 1.3"`, `rpy="-1.57 -1.57 0"` (即 $-\pi/2, -\pi/2, 0$)
  - _解释_：这个关节将坐标系向上移动 1.3 米，并旋转，使得后续的 Z 轴变为水平方向，以便大臂上下摆动。

- **Joint: `J_1`**

  - **类型**: Revolute (旋转关节)
  - **父连杆**: (连接自 J_F0 的输出)
  - **子连杆**: `link_02`
  - **Origin (变换)**: `xyz="0 0 0.35"`, `rpy="0 0 0"`
  - **轴 (Axis)**: Z 轴
  - **限制 (Limits)**: $\pm 3.14$

- **Joint: `J_2`**

  - **类型**: Revolute (旋转关节)
  - **父连杆**: `link_02`
  - **子连杆**: `link_03`
  - **Origin (变换)**: `xyz="0 0.9 0.2"`, `rpy="0 0 0"`
  - **轴 (Axis)**: Z 轴
  - **限制 (Limits)**: $\pm 3.14$

- **Joint: `J_3`**
  - **类型**: Revolute (旋转关节)
  - **父连杆**: `link_03`
  - **子连杆**: `effector`
  - **Origin (变换)**: `xyz="0 0.9 0.05"`, `rpy="1.57 1.57 3.14"` (即 $\pi/2, \pi/2, \pi$)
  - **轴 (Axis)**: Z 轴
  - **限制 (Limits)**: $\pm 3.14$

### 4. 任务总结

"Generate a ROS 2 URDF xacro file (`robot.urdf.xacro`) for a PUMA robot based on the following specifications.

**Links (Geometries):**

1. `base_link`: Cylinder (radius=0.35, length=0.1).
2. `link_01`: Cylinder (radius=0.15, length=1.1).
3. `link_02`: Box (size="0.3 1.2 0.2").
4. `link_03`: Box (size="0.2 1.0 0.1").
5. `effector`: Box (size="0.2 0.2 0.1").

**Joints (Kinematics):**

- `J_0` (Revolute, axis Z, limit +/-3.14): Connects `base_link` to `link_01`. Origin: xyz="0 0 0.1".
- `J_F0` (Fixed): Connects `link_01` to an intermediate link (or directly sets frame for link_02). Origin: xyz="0 0 1.3", rpy="-1.5708 -1.5708 0".
- `J_1` (Revolute, axis Z, limit +/-3.14): Connects from `J_F0` frame to `link_02`. Origin: xyz="0 0 0.35".
- `J_2` (Revolute, axis Z, limit +/-3.14): Connects `link_02` to `link_03`. Origin: xyz="0 0.9 0.2".
- `J_3` (Revolute, axis Z, limit +/-3.14): Connects `link_03` to `effector`. Origin: xyz="0 0.9 0.05", rpy="1.5708 1.5708 3.14159".

Ensure proper simple material colors for visualization. Use basic collision and inertial tags (can be dummy values)."
