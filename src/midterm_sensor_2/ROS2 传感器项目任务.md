# ROS2 传感器项目任务

## 任务背景

您收到了一个传感器,其目的是测量机器人周围的环境并给出一些关于环境的编码信号。您找不到关于该传感器的文档,但您的同事告诉您,该传感器要么给出负数,要么给出正奇数,要么给出正偶数。

您已经完成了大部分工作,了解了传感器的功能,并弄清楚了在项目中如何使用该传感器:

### 传感器数据与机器人行为对应关系

- **如果传感器给出负数** → 机器人需要向前移动 1 个单位 (1.0)
- **如果传感器给出正奇数** → 机器人需要向右转 (90 度)
- **如果传感器给出正偶数** → 机器人需要向左转 (90 度)
